# 🏥 Sistema de Triagem para UBS

Sistema desenvolvido em **Python** com integração a banco de dados **SQL** para otimizar o processo de triagem em Unidades Básicas de Saúde (UBS), permitindo o registro, gerenciamento e análise de pacientes de forma eficiente.

---

## 📖 Descrição

Este projeto simula um sistema de triagem utilizado em UBS, com foco na organização dos dados dos pacientes e apoio à tomada de decisão através de análise de dados.

A aplicação implementa operações completas de **CRUD** e utiliza a biblioteca **Pandas** para manipulação e análise das informações.

---

## 🚀 Tecnologias

* Python
* SQL (PostgreSQL)
* Pandas

---

## ⚙️ Funcionalidades

* Cadastro de pacientes
* Consulta de registros
* Atualização de dados
* Remoção de pacientes
* Análise de dados com Pandas
